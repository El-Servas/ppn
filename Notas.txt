''''

Los Parámetros Valor se usan directamente cuando se necesiten.

''''

Los Parámetros Mapeados se deben de Resolver antes de poderse usar.

Para Resolverlos, se debe de saber el Número de Columna que les corresponde.

Para saber el Número de Columna hay que buscar su nombre de columna en los encabezados del archivo de datos.
Se aprovecha este proceso para determinar el índice de inicio de resultados, y escribir las etiquetas.


''''

En cada fila, se deben de resolver los parámetros mapeados

Se recorren las dos listas de Parámetros, pegando el valor (directo o resuelto) en el rango correspondiente (Parámetro)

Se recorre la lista de Resultados, usando el índice de resultados para plasmarlos.
